Below are a few important links for this project which include the link to the source code, the research paper, the intermediary datasets (before merging) and the final data sets (after merging). The intermediary datasets are nothing but our samples (userid, businessid) combinations that we collected in chunks and hence are present in separate files (JSON format). These files do not have some of the fetures that we include from the yelp data set which we add later when merging. We have provided these as this intermediary data is the outcome of the first two parts of code which are the Feature formation component and Personalized Page Rank component.
